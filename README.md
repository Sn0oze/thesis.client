## installng dependincies
Running this application requires a recent installation of Node.js (https://nodejs.org/en/)
To install the depedencies open a console in the insights directory and type `npm install`

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
